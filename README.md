# Project Files

Personal utility scripts for development.

---

## Usage

```bash
curl -L https://github.com/Holmose/songge-system-install/raw/main/setup.sh -o setup.sh
bash setup.sh
```

Then follow the on-screen instructions.

- `config.json` - Configuration data
- `docs/` - Documentation
